Techno Store
Мой первый проектbackend интернет-магазина техники на Spring Boot с встроенной банковской системой.

🛠 Стек технологий
Java 17+
Spring Boot 3.x (Web, Data JPA)
Maven
Spring Data JPA / Hibernate
REST API
Lombok
База данных: H2 / PostgreSQL (настраивается в application.properties)
📦 Основные модули
| Модуль | Описание | |--------|----------| | Product | CRUD товаров магазина | | Cart | Управление корзиной покупок | | Warehouse | Учёт остатков на складе | | MiniATM | Банковская система (депозит, снятие, оплата) |

🔧 Функции
Товары (Product)

Получение списка товаров (пагинация)
Получение/создание/редактирование/удаление товара
Корзина (Cart)

Создание корзины
Добавление/удаление товаров
Очистка корзины
Просмотр содержимого
Склад (Warehouse)

Управление остатками
Уменьшение количества при заказе
Банковская система (MiniATM)

Проверка баланса (рубли/доллары)
Пополнение счёта
Покупка товара по ID или из корзины
